# Liquidity Happy Hours

Public beta GitHub Pages build for Liquidity Happy Hours, a nonprofit research organization in formation, with AI Data Centers as the first public issue lab.

This is a soft public review tool. It is not legal advice, not an expert report, and not a commission-ready evidentiary record.

Core product promise: pick your role, clear one ledger, leave with a better question.

## Entry

Open `index.html`, then use the in-page hash routes:

- `#/` — role-first Liquidity Happy Hours landing page
- `#/about` — organization mission and operating rules
- `#/platform` — money-view civic platform and role paths
- `#/projects` — issue lab overview
- `#/news` — update ledger and launch-readiness status
- `#/donate` — early support and pledge contact path
- `#/start-here` — role-first AI Data Centers entry
- `#/pocket-card` — five hearing questions and phone card
- `#/lite-review` — hearing question, document requests, and ledger status
- `#/claim-to-ledger` — exact-source claim analyzer
- `#/county-x` — fictional 300 MW walkthrough
- `#/reporter-kit` — deadline mode and headline checker
- `#/evidence` — public-first evidence cards and advanced table
- `#/social-assets` — share cards, captions, alt text, and downloads
- `#/launch-readiness` — public beta gate status

See `route-manifest.json` for the complete route manifest.

## Bundled Downloads

The `research-package/` folder contains the RC3 PDF artifacts and RC2 workbook files linked from the site.

The PDFs are RC3 because public caveats, role framing, and beta metadata were revised after earlier release candidates. The XLSX workbooks remain RC2 because the workbook formulas and certificate structure did not require the RC3 text-only PDF revisions.
